HTML Page Structure

The foundation of every webpage is a clean HTML structure. Here’s the simplest example:

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>My Page</title>
</head>
<body>
  <header>Site Header</header>
  <main>Main Content</main>
  <footer>Footer</footer>
</body>
</html>

CSS Flexbox Centering

Center elements horizontally and vertically with just a few CSS rules.

.center {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 200px;
}

Responsive Grid Layout

A simple grid that changes from 3 columns to 1 column on smaller screens.

.grid {
  display: grid;
  gap: 16px;
}
.grid.cols-3 {
  grid-template-columns: repeat(3, 1fr);
}
@media (max-width: 900px) {
  .grid.cols-3 {
    grid-template-columns: 1fr;
  }
}

HTML Forms

Create interactive forms to collect user input.

<form>
  <label for="name">Name:</label>
  <input id="name" type="text" required>
  <label for="email">Email:</label>
  <input id="email" type="email" required>
  <button type="submit">Submit</button>
</form>

CSS Media Queries

Make your site adapt to different screen sizes.

@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
